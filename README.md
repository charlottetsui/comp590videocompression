For assignment 1, I completed it by using temporal prediction. Each pixel is predicted using the same pixel from the previous frame. The residual between the actual and prediced is arithmetically encoded using 256 contexts determined by the prior frame's pixel value at that location. This works since pixels at different brightness have different temporal variance patterns. As a result, even though we started off with a compression ratio of 2.36, I got it to a compression ratio of 2.74.
